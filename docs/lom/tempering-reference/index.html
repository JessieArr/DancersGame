Primary Material Reference:

Key:

  • GC: Growth Control
  • Sh: Sharp (offense)
  • He: Heavy (offense)
  • Fo: Force (offense)
  • Te: Tech (offense)
  • St: Strike (defense)
  • Sl: Slash (defense)
  • Pi: Pierce (defense)
  • MD: Magic (defense)
  • Wi: Wisp (element)
  • Sh: Shade (element)
  • Dr: Dryad (element)
  • Au: Aura (element)
  • Sa: Salamander (element)
  • Gn: Gnome (element)
  • Ji: Jinn (element)
  • Un: Undine (element)
